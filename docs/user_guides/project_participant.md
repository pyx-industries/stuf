# Project Participant (PP) Guide

## Uploading Files

1. **Access the Upload Portal**
   - Click the URL link in the invitation email sent by the Trust Architect
   - This email contains:
     - A simple, memorable URL for the upload portal
     - Information about what files you should upload
     - Any required metadata you'll need to provide
   - The portal will display your organization's branding

2. **Authentication**
   - Enter your email address
   - If first time:
     - You must have previously clicked the magic link in the invitation email sent by the Trust Architect
     - This verification step is required before you can proceed
   - Enter the verification code (OTP) sent to your phone

3. **Select Files**
   - Click "Select Files" or drag files to the upload area
   - Select one or more files from your device
   - Files will be validated for size and type

4. **Add Required Metadata**
   - Add a description for the upload
   - Complete all required metadata fields (as specified in your invitation email)
   - **Note: The specific metadata fields you see will depend on what the Trust Architect has configured for this STUF**
   - Metadata fields may include some or all of the following:
     - File type (from a list configured by the Trust Architect)
     - Collection (from a list configured by the Trust Architect)
     - IP ownership declaration:
       - Select from predefined options or enter custom text (depending on configuration)
       - Note that this declaration creates a record of your IP ownership claim
       - If "Don't know" is selected (when allowed), you may need to provide additional information
       - Be aware that this declaration may be legally binding (if configured as such)
     - License conditions:
       - Select from predefined license options or enter custom text (depending on configuration)
       - For certain licenses, you may need to provide additional documentation
       - If "Don't know" is selected (when allowed), you may need to provide additional information
       - This establishes the terms under which your organization is sharing this content
     - Comments (may be required for specific file types, collections, ownership types, or license conditions)
   - For any field with "Other (specify)" option, provide details in the comment field
   - Add any optional metadata that may be helpful
   - Note: All metadata entries are recorded and attributed to you as part of the governance process

5. **Upload Files**
   - Click "Upload" to start the process
   - Progress bar will show upload status
   - Wait for confirmation message

6. **Confirmation**
   - Receive on-screen confirmation
   - Upload complete message will be displayed

## Managing Your Uploads

1. **View Upload History**
   - Log in to the upload portal
   - Navigate to "My Uploads"
   - View status of previous uploads

2. **Upload Again**
   - Use "Upload Again" button for quick access
   - Your authentication will be remembered for a limited time

3. **Get Help**
   - Use "Contact Support" for assistance
